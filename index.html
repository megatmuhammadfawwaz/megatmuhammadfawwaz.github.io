// Platform

SURIA

Attack Surface Management System

2026 🔒 Private Repo

A full stack passive attack surface management platform I designed and built to map and monitor an organisation's internet facing footprint. It aggregates external exposure data from Shodan and seven OSINT sources into one console, then flags exactly what changed between scans.

Highlights
  • Aggregates Shodan and 7 OSINT sources: Certificate Transparency, DNS, TLS, WHOIS/RDAP, Wayback, AbuseIPDB and VirusTotal.
  • A snapshot based change detection engine where every scan is an immutable snapshot, diffed by a service identity key and content fingerprint.
  • Governed scanning with role based access, an approval whitelist and an immutable audit trail.
  • Multi snapshot comparison and evidence export to JSON, Excel and PDF.
Result

2,000+ Assets Monitored · 8 Intelligence Sources Unified

  • FastAPI
  • React
  • TypeScript
  • Shodan
  • OSINT
  • PostgreSQL
// Applied AI

Explainable Deepfake Detection

Final Year Project · Explainable AI For Visual Forensics

2025–2026 · FYP 🔒 Private Repo

An end to end deepfake detection pipeline that doesn't just decide, it shows its reasoning, bringing transparency and forensic trust to AI driven media authentication.

Highlights
  • ResNet-18 (PyTorch) classifier with YOLOv8 automated face extraction.
  • Grad-CAM++ explainability heatmaps that visualise every decision.
  • Deployed on Streamlit for interactive, evidence backed authentication.
  • Validated on a balanced benchmark dataset with strong generalisation.
Result

95.45% Accuracy · 0.9988 AUC-ROC · 87.50 SUS
